# PCB Design – 3-Floor Elevator Controller

This folder contains the PCB design deliverables for the 3-floor elevator controller project.

The design implements a small FPGA-based elevator controller with floor input buttons, status LEDs, a single-digit 7-segment display, JTAG/configuration interface, oscillator, and external regulated 3.3 V power input.

## Project Overview

The PCB is based on a Lattice MachXO2 FPGA and is designed as a simplified controller for a 3-floor elevator system.
